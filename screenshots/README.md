## Naming Convention

- Format: `wubuos-{subject}-{date}.png`
- Date format: `YYYY-MM-DD`
- All images should be PNG, 1024×768 or screen-native resolution
- Keep image size < 100KB (use PNG optimization if needed)

## Linking to These Images

In documentation, use repo-root-relative paths:

```markdown
![Desktop (Win98 theme)](screenshots/wubuos-desktop-2026-07-07.png)
```
